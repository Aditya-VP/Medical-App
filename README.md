# Virtual Doctor Healthcare Android Application

## Overview

A feature-rich healthcare app built in **Java (Android Studio)** using **Material Design** and **SQLite**, enabling users to manage medical appointments, prescriptions, medicine orders, lab test scheduling, and more. Designed for patient-doctor interaction, smart records tracking, and practical telehealth use.

---

## Features

- **User Authentication:** Registration and login with patient/doctor role support.
- **Dashboard:** Easy navigation to all healthcare modules.
- **Appointment Booking:** Schedule appointments with doctors.
- **Medicine Purchase:** Order medicines and track orders.
- **Lab Test Scheduling:** Book lab tests and view schedules.
- **Healthcare Articles:** Read curated articles for health awareness.
- **Prescriptions:** Save, view, and manage medical prescriptions.
- **Order & Appointment History:** Track all past bookings/orders.
- **In-App Notifications:** Get timely reminders for appointments or tasks.
- **SQLite Data Storage:** Persist user and record information locally.

---

## Installation

1. Clone this repository:
    ```
    git clone https://github.com/Aditya-VP/Medical-App.git
    ```

2. Open in **Android Studio** (minimum SDK: API 21+ recommended).
3. Android 7.0+.
4. Build the project and run on emulator/device.

---

## Usage

1. **Register:** Create an account as "Patient" or "Doctor".
2. **Login:** Enter your email/password.
3. **Explore the modules:**
    - Book appointments, order medicines, schedule lab tests.
    - Add/view prescriptions and read health articles.
    - Receive in-app reminders and view your history.
